# 贡献指南

感谢你对本项目的关注！欢迎通过以下方式参与贡献。

## 如何贡献

### 报告 Bug
1. 在 [Issues](https://github.com/Royhoo/write-a-chinesechess-program/issues) 页面搜索是否已有相同问题
2. 若没有，新建 Issue，描述清楚：
   - 复现步骤
   - 期望行为 vs 实际行为
   - 浏览器版本和操作系统

### 提交功能建议
- 在 Issues 中用 `enhancement` 标签描述你的想法
- 说明使用场景和实现思路

### 提交代码（Pull Request）

```bash
# 1. Fork 本仓库
# 2. 克隆你的 Fork
git clone https://github.com/YOUR_USERNAME/write-a-chinesechess-program.git

# 3. 创建功能分支
git checkout -b feature/your-feature-name

# 4. 提交改动
git commit -m "feat: 描述你的改动"

# 5. 推送并发起 PR
git push origin feature/your-feature-name
```

## 欢迎的贡献类型

- 🐛 Bug 修复
- ♟️ 棋子走法规则改进
- 🧠 AI 算法优化（评估函数、搜索优化等）
- 🌐 英文文档翻译
- 📱 移动端适配改进
- 🎨 界面美化

## 代码风格

- 使用 2 空格缩进
- 变量和函数名使用 camelCase
- 关键逻辑请添加注释（中文英文均可）
